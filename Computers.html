Computers!

This is the basics of computers!

RAM

Random Access Memory or RAM is also known as memory. This is not to be confused with storage. RAM is the memory that programs can use.

Hard Disk Drive and Solid State Drives

A drive is a type of storage. This is where you store your documents, music, programs, ect. The difference between a Hard Disk Drive(HDD) and a Solid State Drive(SSD) is that the HDD has moving parts while the SSD does not. SSD's are faster but currently are smaller in storage capacity than HDDs.
